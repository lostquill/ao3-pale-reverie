# Pale Reverie — AO3 Site Skin

**Pale Reverie** layers a dusty rose and cool slate palette over the default AO3 layout, with color-coded tags and a constellation border framing work metadata. The overall aesthetic is desaturated and quiet.

A dark mode version may come in the future.

## Features

- Color-coded tags by type (warnings, relationships, characters, freeforms)
- Icon-based stats display with hover text labels
- Constellation border around work metadata
- Gradient header in muted rose and slate tones
- Responsive layout for tablet and mobile
- All colors defined as CSS variables for easy customization

## Installation

See the [AO3 work](https://archiveofourown.org/works/84166896) for full installation instructions.

## Files

| File | Description |
|------|-------------|
| `main.css` | Base styles — colors, tags, layout, and stats icons |
| `tablet.css` | **Tablet skin** — Layout adjustments for tablets (max-width: 62em) |
| `mobile.css` | **Mobile skin** — Minor resets for smaller screens (max-width: 42em) |
